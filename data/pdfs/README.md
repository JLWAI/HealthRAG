# HealthRAG PDF Document Directory

This directory contains your personal fitness and nutrition documents that are processed by HealthRAG.

## Adding New Documents

1. Copy PDF files to this directory
2. Run the processing script: `python3 process_pdfs.py`
3. The documents will be automatically indexed and available for queries

## Current Documents

Add your fitness and nutrition PDF documents here. Example types:
- Training programs and guides
- Body recomposition guides
- Diet and nutrition resources
- Meal planning guides
- Supplement protocols

## File Formats

**Supported for RAG:**
- PDF files (`.pdf`)

**Stored but not processed:**
- Excel tracking sheets (`.xlsx`)
- ACSM files (`.acsm`)

To enable processing of additional formats, file a feature request or contribute to the project.
